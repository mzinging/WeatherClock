# WeatherClock
一个可以显示天气的小时钟，背景图片会随时间变化
修改城市方法：在城市代码.json中找到城市对应的代码，将WeatherClock.py 第18行url 末尾城市代码替换即可。
在当前城市日出日落时间有特定背景

2021年更新：这个获取天气的API已经挂了，感兴趣的小伙伴可以自己再找一个API重写一下
